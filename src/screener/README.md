# Module 6 - Investment Screener Engine

## Overview

The Investment Screener Engine is a comprehensive screening, filtering, sorting, and ranking system for Nifty 100 companies. It leverages financial KPIs generated by previous modules (1-5) to enable sophisticated investment analysis.

## Features

### ✅ Core Functionality
- **Multi-criteria filtering** with 11 operators (>, >=, <, <=, =, !=, BETWEEN, IN, NOT IN, IS NULL, IS NOT NULL)
- **Unlimited filter combinations** with AND/OR logic
- **Multi-field sorting** (ascending/descending)
- **Company ranking** by any financial metric
- **CSV export** with metadata
- **Custom screen templates** saved to SQLite
- **10 preset screeners** for common investment strategies

### ✅ Supported KPIs

**Valuation Metrics**
- PE Ratio, PB Ratio, PS Ratio

**Profitability Metrics**
- ROE, ROA, ROCE, Net Profit Margin, Operating Profit Margin

**Growth Metrics**
- Revenue CAGR (3yr, 5yr), PAT CAGR (3yr, 5yr), EPS CAGR (3yr, 5yr)

**Cash Flow Metrics**
- Free Cash Flow, FCF Margin, Cash Conversion, Cash Return on Assets

**Leverage Metrics**
- Debt to Equity, Current Ratio, Quick Ratio, Interest Coverage

**Health Scores**
- Overall Score, Rating, Category Scores (Profitability, Growth, Cashflow, Leverage, Efficiency)

**Dividend Metrics**
- Dividend Yield

## Quick Start

### Basic Screening

```python
from src.screener.engine import ScreenerEngine

# Initialize engine
engine = ScreenerEngine()

# Load data from database
engine.load_data()

# Apply filters
filters = [
    {"field": "roe", "operator": ">=", "value": 15},
    {"field": "debt_to_equity", "operator": "<=", "value": 0.5},
]

# Screen companies
result = engine.screen_companies(
    filters=filters,
    sort_by="roe",
    sort_ascending=False,
    rank_by="roe"
)

# Access results
companies = result["results"]
print(f"Found {result['count']} companies")
for company in companies:
    print(f"{company['company_name']}: ROE={company['roe']}, Rank={company['rank']}")
```

### Custom Screen Templates

```python
from src.screener.templates import ScreenTemplateManager

# Initialize manager
manager = ScreenTemplateManager()

# Save a custom screen
result = manager.save_screen(
    name="My Custom Screen",
    description="High ROE, low debt, good growth",
    filters=[
        {"field": "roe", "operator": ">=", "value": 18},
        {"field": "debt_to_equity", "operator": "<=", "value": 0.3},
        {"field": "revenue_cagr_3yr", "operator": ">=", "value": 10},
    ],
    sort_by="roe",
    sort_order="desc",
    rank_by="roe"
)

# Load a custom screen
screen = manager.load_screen("My Custom Screen")

# Use it in screening
result = engine.screen_companies(custom_screen_name="My Custom Screen")

# List all custom screens
screens = manager.list_screens()

# Delete a screen
manager.delete_screen("My Custom Screen")
```

## Filter Operators

| Operator | Description | Example |
|----------|-------------|---------|
| `>` | Greater than | `{"field": "roe", "operator": ">", "value": 20}` |
| `>=` | Greater than or equal | `{"field": "roe", "operator": ">=", "value": 15}` |
| `<` | Less than | `{"field": "pe_ratio", "operator": "<", "value": 20}` |
| `<=` | Less than or equal | `{"field": "pe_ratio", "operator": "<=", "value": 25}` |
| `=` | Equal | `{"field": "sector", "operator": "=", "value": "IT"}` |
| `!=` | Not equal | `{"field": "sector", "operator": "!=", "value": "Energy"}` |
| `BETWEEN` | Between two values | `{"field": "roe", "operator": "BETWEEN", "value": 15, "value2": 25}` |
| `IN` | In list | `{"field": "sector", "operator": "IN", "value": ["IT", "Banking"]}` |
| `NOT IN` | Not in list | `{"field": "sector", "operator": "NOT IN", "value": ["Energy"]}` |
| `IS NULL` | Is null/empty | `{"field": "dividend_yield", "operator": "IS NULL"}` |
| `IS NOT NULL` | Is not null/empty | `{"field": "dividend_yield", "operator": "IS NOT NULL"}` |

## API Reference

### ScreenerEngine

#### Main Methods

- `load_data()` - Load financial data from database
- `apply_filters(filters, logic="AND")` - Apply filter conditions
- `sort_results(sort_by, ascending=True)` - Sort results
- `rank_companies(rank_by, ascending=False)` - Rank companies
- `screen_companies(...)` - Complete screening pipeline
- `export_results(...)` - Export to CSV

#### Convenience Methods

- `screen_by_preset(preset_id, ...)` - Screen using preset
- `screen_by_custom(screen_name, ...)` - Screen using custom template
- `get_statistics()` - Get screening statistics

### ScreenTemplateManager

- `save_screen(name, filters, ...)` - Save custom screen
- `load_screen(name)` - Load custom screen by name
- `load_screen_by_id(id)` - Load custom screen by ID
- `delete_screen(name)` - Delete custom screen
- `list_screens()` - List all custom screens
- `get_screen_count()` - Get count of custom screens

### ScreenerExporter

- `export_results(results, ...)` - Export to CSV
- `export_filtered_results(results, filters)` - Export with filter metadata
- `export_ranked_results(results, rank_field)` - Export with ranking metadata

## Database Schema

### screen_templates Table

```sql
CREATE TABLE IF NOT EXISTS screen_templates (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL UNIQUE,
    description TEXT,
    filters TEXT NOT NULL,  -- JSON array of filter conditions
    sort_by TEXT,
    sort_order TEXT DEFAULT 'asc',
    rank_by TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
